# Sokoban
Juego de Sokoban en Python con Tkinter. Controlás al jugador con las flechas del teclado para empujar 3 cajas hacia sus destinos en un mapa de 10x10. El código valida colisiones con paredes o bloqueos, registra tus pasos en tiempo real y muestra un aviso de victoria cuando lográs ubicar todas las cajas en su lugar.
